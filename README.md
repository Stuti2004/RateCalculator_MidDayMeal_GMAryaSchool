## 🖥 How to Use

1. **Select the month**

2. **Select the class group**

3. Click **Make Table**

4. Enter **daily student attendance**

5. The system will automatically calculate:
   - Total students
   - Grain amount
   - Wheat usage
   - Rice usage

6. Use the available options:
   - **Excel Export** → Download the register as an Excel file
   - **Print** → Print the full sheet on one page
